# Insert pages into an existing PDF file

MS-Edge Copilot:

Got it ‚Äî you want to insert pages into an existing PDF using the Linux command line.
Here are three practical ways to do it:

1. Using `pdftk` (simple and widely available)

```bash
pdftk original.pdf cat 1-180 newpages.pdf original.pdf cat 181-end output merged.pdf
```

- 1-180 ‚Üí pages before insertion point

- `newpages.pdf` ‚Üí the PDF containing pages you want to insert

- 181-end ‚Üí remaining pages after insertion
2. Using `qpdf` (lightweight and fast)

```bash
qpdf --empty --pages original.pdf 1-180 newpages.pdf original.pdf 181-z -- merged.pdf
```

`-z` automatically means ‚Äúlast page‚Äù in `qpdf`.

3. Using `pdfunite` (from `poppler-utils`)
   First, split the original PDF:

```bash
pdftk original.pdf cat 1-180 output part1.pdf
pdftk original.pdf cat 181-end output part2.pdf
```

Then merge:

```bash
pdfunite part1.pdf newpages.pdf part2.pdf merged.pdf
```
